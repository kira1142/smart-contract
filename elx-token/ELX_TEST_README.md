# How to run ELXTokenTest.js
- [ELXTokenTest.js](test/ELXTokenTest.js) is the test script for ELXToken on testnet using file [elx_sc_test_flattened.sol](contracts/flatten/elx_sc_test_flattened.sol)
## Setup dependencies
- Require: Nodejs
```shell
npm i
```
## Run test
```shell
npx hardhat complie
npx hardhat test
```
